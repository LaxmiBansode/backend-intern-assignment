## **How to Take Screenshots (Windows)**

### **Step 1 — Take Screenshots**
- Press `Windows + Shift + S` → Select area → Screenshot saved to clipboard
- Paste in any image editor or directly in Google Form

### **Screenshots You Need:**

**1. Registration Page**
- Go to `http://localhost:3000` → Click "Register"
- Screenshot the form

**2. Password Length Error**
- Type password less than 6 chars → Click Register
- Screenshot the error message

**3. Registration Success**
- Register with valid details
- Screenshot success message

**4. Login Page**
- Screenshot the login form

**5. Login Success**
- After login → Dashboard view
- Screenshot showing "Welcome" message

**6. Create Task**
- Fill task form → Before clicking Create
- Take screenshot

**7. Task Added**
- After clicking Create → Task appears in list
- Screenshot showing new task

**8. Edit Task**
- Click Edit button → Pre-filled form
- Screenshot

**9. Delete Task**
- Click Delete → Confirmation dialog
- Screenshot

## **Quick Way to Paste Screenshots:**
If Google Form accepts images, just:
1. `Windows + Shift + S`
2. Select area
3. `Ctrl + V` directly in the form
